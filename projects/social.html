Overview

A full-stack product with a reactive frontend and a scalable backend, designed to support streaming content, search, and recommendations.

What I Built

  • Frontend experience in React with a Node.js/Express backend integration.
  • Real-time data pipeline components with Kafka and Spark for streaming features.
  • Search and face-matching workflows backed by ChromaDB.

Features

  • Reactive Frontend with React and Node.js/Express backend.
  • Scalable infrastructure on AWS (EC2, RDS, S3).
  • Real-time data processing with Apache Kafka and Spark.
  • Advanced search/matching using ChromaDB for face-matching.

Tech Stack

  • React, Node.js, Express
  • AWS (EC2, RDS, S3)
  • Apache Kafka, Apache Spark
  • ChromaDB, MySQL
