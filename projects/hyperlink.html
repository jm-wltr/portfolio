Hyperlink Extractor

MV3 Chrome Extension for efficient link management.

Developed tool to copy text with hyperlinks to clipboard with the option to automatically add a ChatGPT prompt.

Overview

A browser-side workflow that turns selected content into clean, reusable link data and optional prompt-ready output.

Workflow

  • Interactive sidebar for settings and extraction history.
  • Clipboard integration for copying linked text or prompt-augmented output.
  • Manifest V3 architecture built with the Chrome Extension API.

Functionality

  • Interactive sidebar with settings and extraction history.
  • Seamless integration with clipboard and ChatGPT prompts.
  • Built using Manifest V3 standards.

Tech Stack

  • JavaScript (Chrome Extension API)
  • HTML/CSS (Sidebar UI)
  • Manifest V3
