Giggle

Distributed web search engine built for scale and performance.

Core Features

  • Distributed Crawler: Handles millions of pages with politeness and efficiency.
  • Inverted Indexer: Parallelized indexing using Spark and distributed KVS.
  • Ranking Engine: PageRank-inspired algorithms for relevance.

Tech Stack

  • Java
  • AWS EC2
  • Distributed KVS
  • Spark
  • Kafka

Achievements

Successfully crawled and indexed a large subset of the web, handling edge cases like trap loops and slow servers.
