Sonic Tomography AI

End-to-end PyTorch pipeline for sonic tomographic reconstruction of historic masonry walls.

Overview

Developed a learning-based approach to reconstruct the internal structure of walls using acoustic ray data. Achieved 93% accuracy on historical samples.

Architecture

  • Data Ingestion: SpecFEM simulation data processing.
  • Model: Deep CNN for pixel-wise material classification.
  • Pipeline: Reproducible training and inference with PyTorch.

Technologies

  • PyTorch
  • Python
  • SpecFEM
  • Signal Processing
